Welcome to

The Chain

A game to test your musical abilities!

Open your ears and let the fun begin!

Game rules:

  • Press the "Easy" button to switch the game between easy and hard mode

  • Press "Play" to start the game

  • You will hear a note, and the corresponding key will be highlighted in easy mode

  • When the sequence is over, try and remember it and repeat it back using the keys!

  • For each sequence that you repeat correctly, the note sequence will increase by one key and the level will increase

  • You will get 1 point for each correctly repeated sequence

  • If you repeat the sequence incorrectly, you lose your points, and you return to level 1

  • Good Luck!
